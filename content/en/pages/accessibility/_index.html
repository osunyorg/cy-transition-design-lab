Transition Design Lab is committed to making its website accessible in accordance with Article 47 of Law No. 2005-102 of February 11,2005.

This accessibility statement applies to the website transitiondesign. labo. cyu. fr.

notes: >- alt: >- credit: >- - kind: heading title: >- Establishment of this Accessibility Statement slug: "etablissement-de-cette-declaration-d-accessibilite" position: 4 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

This statement was established on 05/06/2024.

Technologies used for the website's development:

  • HTML5
  • CSS
  • JavaScript
  • Hugo

User agents and assistive technologies used to verify accessibility:

  • NVDA
  • VoiceOver

Accessibility verification was conducted through manual testing, assisted by the following tools:

  • Accessibility Insights for Web
  • ArcToolkit
  • Assistant RGAA
  • Axe DevTool
  • Color Contrast Analyser
  • eAccessibility (PDF accessibility checker)
  • Component Inspector
  • Web Developer Toolbar
  • WCAG Contrast Checker
  • W3C HTML Validator
notes: >- alt: >- credit: >- - kind: heading title: >- Testing Environment slug: "environnement-de-test" position: 5 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Content rendering verifications were conducted based on the reference combination provided by the RGAA baseline, with the following versions:

  • On MacOS computer with Google Chrome and VoiceOver
  • On MacOS computer with Safari and VoiceOver
  • On Windows computer with Firefox and NVDA
  • On Android mobile with Google Chrome and TalkBack

Si vous avez signalé au responsable du site internet un défaut d'accessibilité qui vous empêche d'accéder à un contenu ou à un des services du portail et n'avez pas obtenu de réponse satisfaisante, vous êtes en droit de:

  • Écrire un message au Défenseur des droits (formulaire en ligne);

  • Contacter le délégué du Défenseur des droits dans votre région;

  • Envoyer un courrier postal (gratuit, ne pas mettre de timbre) à cette adresse: Défenseur des droits - Libre réponse 71120 - 75342 Paris CEDEX 07.
